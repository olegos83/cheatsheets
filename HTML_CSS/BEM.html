BEM

Block • Element • Modifier

B
Block
A standalone, reusable page component
E
Element
A part of the block without standalone meaning
M
Modifier
Changes appearance, behavior or state of the block/element

BEM basic principles in HTML/CSS

Block

  • Block is a standalone, reusable page component.
  • Block can be nested to another block or element.
  • To make block more reusable try not to use margins and paddings in its style.
  • Name each block semantically to represent its functionality.
  • CSS class naming pattern: block-name.

Element

  • Element is always a part of the block.
  • Element must not be used outside the block. It has no standalone meaning and is semantically tied to its block.
  • Element can be nested to other element, but it always belongs to its block (not parent element).
  • Element is optional block part and not all blocks have elements.
  • CSS class naming pattern: block-name__elem-name.

Modifier

  • Modifier is used to change appearance, behavior or state of the block/element.
  • Modifier is an extra class name which you add to a block/element DOM node.
  • Add modifier class only to block/element you wnat to modify after its original class name.
  • CSS class naming pattern: block-name__elem-name_mod-name.
